# **App Name**: Agency Amatrix

## Core Features:

- User Authentication: Secure registration/login for clients and professionals with social authentication and KYC onboarding.
- Service Request with Intelligent Search: Enable clients to request services using an intelligent search engine to filter through categories, service locations, and professional profiles. Suggest related keywords or phrases. Uses generative AI to find services a user might need, but did not know to ask for; the LLM is a tool here.
- Dynamic Negotiation Interface: Implement an Indrive-like flow for service negotiation with geolocation (Google Maps/Leaflet) for physical services, booking system and WebRTC video integration for virtual services.
- Escrow Payment System: Secure payment system with escrow protocol: 50% upfront, service execution, client confirmation, remaining 50% payment and automatic commission application.
- Service Catalog: Hierarchical categorization of physical, wellness, education, digital, and entertainment services. Display the different available services.
- Admin Panel: Enterprise-grade admin panel for total CRUD, financial management, churn rate reports, and multi-region scalability.
- User Profiles: Advanced user profiles with video presentations, optimized bios, verification badges, coverage maps, rate selectors, and success counters.

## Style Guidelines:

- Primary color: Neon Green (#00FF41) for success and action.
- Background color: Obsidian Black (#000000) for a luxurious Matrix aesthetic.
- Accent color: Matrix Purple (#2D004D) for depth and hierarchy.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to create a modern and neutral look. Suitable for both headlines and body text.
- Futuristic vector icons aligned with the Matrix aesthetic. Icons to use Neon Green and Matrix Purple to indicate action or depth/hierarchy.
- Bottom navigation for mobile app, sidebar navigation for web. Clean, frictionless design focused on guiding the user to payment.
- Fluid animations to enhance the user experience and create a sense of speed.
